> _ | rtail

Your terminal output in the browser in seconds, using UNIX pipes.

Ready to tail

Copy/paste the examples below and play with the demo above

Install rtail and start pushing lines to our demo server.
You will see them appear right above

$ npm install -g rtail
$ echo "Hello from the website!" | rtail --id rtail.org --host rtail-demo.lukibear.com
⌘ + C

Or just start typing to stdin

$ rtail --id rtail.org --host rtail-demo.lukibear.com --mute
⌘ + C
